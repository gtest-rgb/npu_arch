## 输出格式说明

### 可行性判断标准

| 结论 | 含义 | 后续行动 |
|------|------|----------|
| ✅ 支持 | 场景完全在Passes能力范围内 | 可以直接实施 |
| ⚠️ 有约束 | 场景支持，但需满足特定条件 | 确认约束条件后再实施 |
| ❌ 不支持 | 场景不在Passes能力范围内 | 需要调整方案或使用其他机制 |

### 约束类型说明

| 类型 | 说明 |
|------|------|
| 硬约束 | 必须满足，否则功能失败 |
| 软约束 | 建议满足，超出会降级处理 |
| 性能约束 | 影响性能但不影响正确性 |

### 引用格式

所有判断依据必须使用以下格式引用：
```
依据：`相对路径#章节标题`
```

例如：
```
依据：`.agents/skills/pypto-pass/pypto-pass-module-analyzer/09_ASSIGN_MEMORY_TYPE.md#8-能力边界`
```
